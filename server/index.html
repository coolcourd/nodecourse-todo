Welcome to my Todo Api

This api currently accepts GET, POST, PATCH, and DELETE requests with no authentication

Get

The simple get url for all todos is "https://pacific-escarpment-41302.herokuapp.com/todos"

The simple get url for a specific todo is "https://pacific-escarpment-41302.herokuapp.com/todos/(insert id here)"

POST

To add a todo send a post request with a JSON body to "https://pacific-escarpment-41302.herokuapp.com/todos" (you will need a non-empty string sent in a "text" property on that object)

PATCH

This works the same as POST, you can include a completed property and set it to true to make "completed at" populate with the UNIX timestamp

Delete

To delete a todo just send a delete request to "https://pacific-escarpment-41302.herokuapp.com/todos/(insert id here)"
